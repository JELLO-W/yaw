ROS2 middleware wrapper. This class is a singleton, so it can be instantiated only once. The activate method should be called to initialize the middleware. The deinit method should be called to deinitialize the middleware and destroy all connections. The activate and deinit methods are automatically called when the class is instantiated and when the program exits, respectively.
